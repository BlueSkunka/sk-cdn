Aperçu :

Utilisation d’un son MP3

Son stocké dans : 

    https://tonpseudo.github.io/sk-cdn/sounds/death.mp3

Exemple JavaScript pour StreamElements / OBS : 

    const audio = new Audio(
      "https://tonpseudo.github.io/sk-cdn/sounds/death.mp3"
    );

    audio.volume = 0.7;

    audio.play();

Exemple HTML : 

    <audio controls>
      <source
        src="https://tonpseudo.github.io/sk-cdn/sounds/death.mp3"
        type="audio/mpeg"
      />
    </audio>

Utilisation d’une police

Police stockée dans :

https://tonpseudo.github.io/sk-cdn/fonts/hells-cemetery.ttf

CSS pour l’importer :

@font-face {
  font-family: "HellsCemetery";
  src: url("https://tonpseudo.github.io/sk-cdn/fonts/hells-cemetery.ttf") format("truetype");
}

Exemple HTML :

<p style="font-family: 'HellsCemetery'; font-size:32px;">
  Exemple de texte avec la police Hell's Cemetery
</p>
